### A command-line tool built in **C#** for performing file and system operations.

# Fetures
| Command | Description |
|----------|--------------|
| **cat**  | Displays the contents of a file. |
| **echo** | Prints text or messages to the terminal. |
| **cd**   | Changes the current working directory. |
| **head** | Shows the first few lines of a file. |
| **tail** | Shows the last few lines of a file. |
| **ls**   | Lists files and folders in the current directory. |
| **pwd**  | Prints the current working directory path. |
| **rm**   | Removes a file or directory. |
| **wc**   | Counts lines, words, and characters in a file. |

# How to use

### The program runs like the **.NET** command terminal.
### Make sure to start with             (dotnet run + "command you want to run").
